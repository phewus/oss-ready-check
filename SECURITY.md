# Security Policy

OSS Ready Check is a local command-line tool. It does not intentionally collect, transmit, or store user data.

## Reporting a vulnerability

If you find a security issue, please do not open a public issue with exploit details. Instead, the contact method that is available on the repository profile.

Please include:

- A short description of the issue
- Steps to reproduce it
- Possible impact
- Suggested fix, if known

## Supported versions

The project is currently in early development. Security fixes will target the latest version.
