# Code of Conduct

This project aims to be welcoming and respectful.

## Expected behavior

- Be kind and constructive
- Respect different experience levels
- Give feedback on ideas and code, not people
- Assume good intent when possible
- Help make open source easier for newcomers

## Unacceptable behavior

- Harassment or personal attacks
- Discriminatory language
- Intimidation or threats
- Repeatedly disruptive behavior
- Sharing private information without permission

## Reporting

If you see behavior that makes the project unsafe or unwelcoming, please open a private contact channel with the maintainer if available. Public reports should avoid exposing private personal information.
